# Cartella save

Questa cartella contiene i salvataggi ufficiali dei turni Capri Blu.

File principale:

- `turni-attuali.json`: copia ufficiale più recente da usare come base comune.

Procedura manuale sicura:

1. Aprire la pagina turni.
2. Modificare i turni normalmente.
3. Usare il pulsante `Scarica backup ufficiale`.
4. Rinominare o caricare il file scaricato come `save/turni-attuali.json` dentro GitHub.

Nota: la pagina usa ancora il salvataggio locale del browser come cache veloce. La cartella `save` serve come archivio ufficiale nella repository.
